# Crevasse 🎣

A clean, data-driven fishing app built with Expo and React Native.

Crevasse helps you track fishing spots, log sessions, and understand when and how to fish using real environmental data like tides, wind, and moon phase.

---

## ✨ Features

### 📍 Spot Management
- Save fishing locations with map-based pin selection  
- Categorize spots (beach, jetty, rocks, estuary)  
- View coordinates and notes  

### 🎣 Session Logging
- Log catches with species, bait, and notes  
- Automatically captures:  
  - Tide state  
  - Wave height  
  - Wind speed  
  - Pressure  
  - Temperature  
  - Moon phase  

### 🌊 Fishing Conditions
- Real-time condition scoring (Good / Fair / Poor)  
- Based on:
  - Tide movement  
  - Wind conditions  
  - Moon phase  

### 🎯 Tackle Recommendations
- Suggested bait types  
- Recommended sinker weight (grams)  
- Wave force awareness  

### 🧠 Personal Insights
- Best bait by spot  
- Top species caught  
- Best tide conditions  
- Best moon phase  
- Wind-based performance patterns  

---

## 🛠 Tech Stack

- React Native (Expo)  
- Expo Router  
- AsyncStorage  
- Open-Meteo API (weather & marine data)

## 🚀 Getting Started

### Install dependencies
npm install

### Start the app
npx expo start

### Run on device
- Scan QR with Expo Go (Android/iOS)  
- Or run on emulator
